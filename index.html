Parameters

Starting Sequence
i
Comma-separated pitch classes (0-11) for the initial set.
Pivot On First Note
i
When ON, pitch inversions pivot around the phrase's own starting note rather than universally pivoting around C. This preserves the original register.
Octave Displacement
i
When ON, each note is randomly shifted one octave up (+12), one octave down (-12), or stays in its base octave. Creates a wider, glitchier melodic gesture.
Euclidean Rhythm
i
When ON, dynamically distributes Notes (Hits) as evenly as possible across a set number of Steps, creating syncopated rhythms.
